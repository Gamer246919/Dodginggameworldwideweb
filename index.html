Dodging Game

Download and play the Dodging Game on your device.

How to Download and Play

For Windows & Mac

  1. Click the download button below.
  2. Extract the ZIP file to a folder.
  3. Open the extracted folder and run dodginggame.py using Python.

For Android

  1. Download the PyDroid 3 app from Play Store.
  2. Click the download button below and extract the ZIP file using any file manager.
  3. Open PyDroid 3, navigate to the extracted folder, and run dodginggame.py.

For iOS

  1. Download the Pyto app from the App Store.
  2. Download and extract the ZIP file.
  3. Open Pyto, navigate to the extracted folder, and run dodginggame.py.
